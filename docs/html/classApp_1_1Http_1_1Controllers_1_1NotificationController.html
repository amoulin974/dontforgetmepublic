◆ getDetails()

getDetails ( )

Retrieve detailed notification information.

This method performs several joins between the tables effectuer, users, reservations, activites, and entreprises to gather comprehensive details. Additionally, it executes subqueries to obtain the first notification's ID, category, state, and delay (before notification) associated with each reservation.

Renvoie
\Illuminate\Http\JsonResponse JSON response containing the detailed notification data.
